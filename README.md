##  [DEV] Deployable Hegic Contracts
https://github.com/stacksideflow/contracts-v888

## Development
Compile contracts (assumes you have `solc` installed):
```bash
truffle compile
```
Migrate:
```bash
truffle migrate
```

Start UI:
```bash
lein build-css
lein repl
(start-ui!)
# go to http://localhost:4177/
```
